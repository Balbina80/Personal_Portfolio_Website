## 🎯 Project Overview

BJ Studio is a personal portfolio website designed and developed to present my web design services for small businesses, personal brands and growing local companies.

The purpose of this project was to create a clean, modern and professional online presence that:

- builds trust with potential clients
- showcases real client projects
- highlights services in a clear and simple way
- supports enquiries and future collaborations

### AJ Fitness – Personal Trainer Website
